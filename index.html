Cпособи зробити git add

git add <список файлів> - git add file1 file2

git add . - додати всі файли до поточної папки

git add *.css - додати всі файли до поточної папки зрозширенням .css

git add someDir/*.css - додати всі файли в папці someDir з розширенням .css

git add someDir/ - додати всі файли в папці someDir

git add “*.css”-додати всі файли в проекті з розширенням .css
